Methods

Message Content Analysis:
We first generated (1) 20 iterations of various hate speech related ideas relating to antisemitic sentiment using a large language model. These examples were (2) used as proxies to represent the way a person might coherently express an antisemitic idea online. We then (3) created a custom semantic search index for all comments made to a single online forum conversation/comments made to a single submission to that forum.
The examples provided and reported on in this report were identified using our semantic search function and have not been altered or summarized in any way using a large language model. All data, thus, is human in origin and not "hallucinated".

Influence/Convergence Analysis:
Rosen and Dale 2023 proposed an unsupervised approach for analyzing linguistic convergence in datasets. Their definition is such that when you can recover more of the information about an earlier text from reading a second text, there is evidence of "convergence"--of the later text borrowing ideas from the earlier text. This can be used to assess the profundity of influence that prior texts have on subsequent texts. This analysis forms the statistical backbone of our influence analytics reporting in the last section, and provides direct insight into how influential the ideas in comments are based on whether those same ideas propagate in later comments.

Message Influence Analysis

When a particular message is introduced to a thread it can have multiple effects. One of those effects is to trigger convergence in subsequent comments. Convergence is the tendency for individuals to adopt similar ways of expressing themselves and their ideas.

In the following charts, we take the identified hate espousing messages authored by individuals and track which comments converge with them. Each node represents a specific comment made in the conversation. Each node is labeled with the name of the user who wrote that comment.

Two relationships between comments are considered. When a later comment is a direct reply to an older comment, then "is_parent" is true (i.e. the older comment is the parent to the later comment). When "is_sibling" is true, both the older comment and the later comment are written in response to the same parent comment. Previous research has shown that there are differences in where and how hateful content tends to cluster in online discussions, with hateful authors tending to respond in the replies to the same comment in what some researchers have dubbed "hate parties" (Rea et al. 2024; Cinelli et al. 2022)
